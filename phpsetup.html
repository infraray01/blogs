How to Upload Files and MySQL in AwardSpace!

Follow these steps:

Step 1: Upload Files via File Manager

  1. Log in to your AwardSpace account and go to the Control Panel.
  2. Navigate to File Manager.
  3. In the File Manager, open the htdocs folder. This is the root folder for your website.
  4. Click on the Upload button and upload all your website files (HTML, CSS, JS, PHP, etc.).
  5. Once uploaded, ensure your main file (e.g., index.html or index.php) is in the root directory.

Step 2: Import Your MySQL Database

  1. In the Control Panel, go to MySQL Databases.
  2. Click on phpMyAdmin to open the database management tool.
  3. Select your database from the left sidebar (e.g., 4531440_login).
  4. Click on the Import tab at the top.
  5. Click the Choose File button and select your SQL file (e.g., database.sql).
  6. After selecting the file, click Go to import the database.
  7. Once the import is complete, verify that the tables have been created successfully.

Step 3: Update Database Connection Details

In your PHP files (like db.php), update your database connection details:

$servername = "fdb1029.awardspace.net";
$username = "4531440_login";
$password = "your_actual_password";
$dbname = "4531440_login";

$conn = new mysqli($servername, $username, $password, $dbname);
if ($conn->connect_error) {
    die("Connection failed: " . $conn->connect_error);
}

Ensure these values match your AwardSpace database credentials.

Step 4: Test Your Website

  1. After uploading your files and importing your database, visit your domain or subdomain (e.g., yourdomain.atwebpages.com).
  2. Verify that your website is working as expected.
  3. If there are any issues, check your PHP code for errors or misconfigurations.
